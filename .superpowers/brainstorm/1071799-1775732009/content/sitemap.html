Estrutura do site — qual abordagem?

Escolha como organizar as páginas institucionais e os guias

A

Site institucional separado dos cursos

Home pública robusta + páginas estáticas. Cursos ficam em /courses como já estão. Guias são páginas web independentes.

/ → home institucional (vitrine TDS)
/sobre → sobre o programa
/municipios → 36 municípios
/courses → catálogo de cursos (existente)
/guia-aluno → guia beneficiário
/guia-tutor → guia instrutor
/guia-gestor → guia gestor
B

Tudo integrado ao LMS

Home aprimorada + guias como cursos/aulas dentro do próprio LMS. Gestor e tutor têm dashboards. Mantém tudo em um sistema.

/home-tds → home melhorada
/courses → catálogo
/batches → turmas
[curso] "Guia do Aluno" → aulas de onboarding
[curso] "Guia do Tutor" → aulas de capacitação
/guia-gestor → web page com métricas
C

Híbrido — landing page + guias como web pages simples

Landing page pública institucional. Guias como páginas web dentro do Frappe (acessíveis sem login). Recomendado: máximo impacto, mínima complexidade.

/ → landing page TDS (vitrine)
/sobre → sobre o TDS
/cursos → = /courses
/municipios → cobertura territorial
/guia-aluno → guia simples (sem login)
/guia-tutor → guia completo (sem login)
/guia-gestor → painel / indicadores

💡 Dica: No Frappe LMS, Web Pages são criadas em Website → Web Page e ficam disponíveis como rotas públicas sem necessidade de login.
