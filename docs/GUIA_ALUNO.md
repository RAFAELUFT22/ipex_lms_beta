# Manual do Aluno — TDS

Bem-vindo ao Território de Desenvolvimento Social (TDS). Sua jornada de aprendizado acontece principalmente pelo WhatsApp e pelo nosso portal web.

## 1. Como Acessar
1. Envie uma mensagem para o número do TDS no WhatsApp.
2. Você receberá um link ou um código de 6 dígitos.
3. Insira o código no portal (`https://portal.ipexdesenvolvimento.cloud`) para acessar suas aulas.

## 2. O Tutor IA (WhatsApp)
Você pode tirar dúvidas sobre o conteúdo do curso diretamente no WhatsApp:
- Envie sua pergunta (ex: "O que é agrofloresta?").
- Se estiver em um grupo, use o comando: `/duvida [sua pergunta]`.
- Para receber um resumo da última lição, use: `/resumo`.

## 3. Certificado
O certificado é gerado automaticamente após a conclusão de 100% das lições. Você pode baixá-lo no portal web.

## 4. FAQ
- **Meu código expirou:** Basta solicitar um novo código na tela de login do portal.
- **Não consigo ver o curso:** Verifique se seu número de WhatsApp é o mesmo utilizado na inscrição.
